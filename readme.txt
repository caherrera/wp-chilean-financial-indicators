== Description ==

== Installation ==

1. Upload the entire `wp-chilean-financial-indicators` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

=== Alternative using git ===

1. Go to plugin folder `/wp-content/plugins/`.
1. Add a new submodule using this command ``` $ git submodule add git@github.com:caherrera/wp-chilean-financial-indicators.git ```
1. Activate the plugin through the 'Plugins' menu in WordPress.
